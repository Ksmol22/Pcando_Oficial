# PCando Oficial - GitHub Pages

## 🚀 Aplicación Desplegada

Esta aplicación está configurada para desplegarse automáticamente en GitHub Pages.

### URLs de Acceso:
- **Aplicación Principal**: [https://ksmol22.github.io/Pcando_Oficial/](https://ksmol22.github.io/Pcando_Oficial/)
- **Demo Independiente**: [https://ksmol22.github.io/Pcando_Oficial/demo.html](https://ksmol22.github.io/Pcando_Oficial/demo.html)

### Estado del Despliegue:
- ✅ Archivos de construcción generados en `/dist`
- ✅ Workflow de GitHub Actions configurado
- ⚠️ **PENDIENTE**: Habilitar GitHub Pages en Settings → Pages → Source: GitHub Actions

### Instrucciones de Configuración:

1. **Ir a la configuración del repositorio**:
   - Ve a: https://github.com/Ksmol22/Pcando_Oficial/settings/pages
   
2. **Configurar la fuente**:
   - En "Source", seleccionar **"GitHub Actions"**
   - Guardar los cambios
   
3. **Verificar el despliegue**:
   - Ir a la pestaña "Actions" para ver el progreso
   - Una vez completado, la aplicación estará disponible en las URLs de arriba

### ¿Qué incluye la aplicación?

- 🖥️ **Configurador de PC completo**
- 🎮 **Componentes reales** (CPU, GPU, RAM, Almacenamiento)
- 💰 **Cálculo de precios en tiempo real**
- ✅ **Validación de compatibilidad**
- 💾 **Almacenamiento local** de configuraciones
- 📱 **Diseño responsive**

---

**Nota**: Si ves un error 404, significa que GitHub Pages aún no está habilitado. Sigue las instrucciones de arriba para activarlo.
